## L'idée
* Le site propose les services d'un centre de formation en informatique qui existe sur Paris.
* mon idée était de proposer une maquette pour un site d'offre de services.
* J'ai choisi de rédiger le site en anglais.

## les données de la map
* Les données exploitées pour les marqueurs viennent de l'opendata de la ville de Paris.
https://opendata.paris.fr/explore/dataset/belib-points-de-recharge-pour-vehicules-electriques-disponibilite-temps-reel/information/?disjunctive.statut_pdc&disjunctive.postal_code
* J'ai choisi le jeu de données portant sur les bornes de recharge des véhicules électiques. 
* Les données ont été filtrées sur les codes postaux correspondant à ceux des antennes du centre de formation (75012 et 75016), dans le but de limiter le nombre de points.
